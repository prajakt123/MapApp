Map App
========
This application will showcase the features of kony map widget.

# Purpose
This app shows the nearby ATM locators showcasing the features of Kony map widget.


# Supported platforms:
**Mobile**
 * iPhone
 * Android
 * BlackBerry
 * Windows Phone 7.5
 * Windows Phone 8
 * SPA iPhone
 * SPA android

**Tablet** 
 * iPad
 * Android Tab
 * Windows8
